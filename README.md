# Structure
* [/ui](ui) - standalone npm package
* [/app-extension](app-extension) - Quasar app extension

# Compile
.../{root}> cd ui
.../{root}/ui> yarn build

# Run
.../{root}> cd ui/dev
.../{root}/ui/dev> quasar dev 

# Publish
.../{root}> cd ui
.../{root}/ui> yarn publish

# Install
.../{proj-root}> npm install @interopshop/quasar-ui-client-controls@x.x.x
