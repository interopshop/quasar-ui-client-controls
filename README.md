# Structure
* [/ui](ui) - standalone npm package
* [/app-extension](app-extension) - Quasar app extension

# Compile
* .../{root}> cd ui
* .../{root}/ui> yarn
* .../{root}/ui> yarn build

# Run
* .../{root}/ui> cd dev
* .../{root}/ui/dev> quasar dev 

# Publish
* .../{root}/ui/dev> cd ../
* .../{root}/ui> yarn publish

# Install
* .../{proj}/{root}> npm install @interopshop/quasar-ui-client-controls
